HTML : Hypertext Markup Language

하이퍼텍스트는 사용자의 선택에 따라 특정한 정보화 관련된 부분으로 이동할 수 있는 조직화된 문서

문자를 동적으로 연결, 사용자의 선택에 따라 특정한 정보, 관련된 부분으로 이동할 수 있게 조직화 표준화된 문서

Hyper+Text (활동적, 이동적+문자) : 문자를 동적으로 연결, 사용자의 선택에 따라 특정한 정보, 관련된 부분으로 이동

Tag : HTML 페이지에서 Elemnet 객체(?)를 만드는데 사용 : 사용할 수 있는 Tag는 W3C 재단에서 표준 제정.

Attribute : Tag 에 추가정보를 부여.

주석 : 프로그래밍 언어에서는 프로그램 실행에 영향을 미치지 않고 설명을 목적으로 하는 코드
